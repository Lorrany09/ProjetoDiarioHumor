# Diario do Humor
Uma aplicação web interativa para registro diário de emoções, permitindo que os usuários expressem seus sentimentos através de emojis e textos, com armazenamento local no navegador.

----
## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)
* Web Storage API (localStorage/sessionStorage)
* Google Fonts
----
## Estrutura do Projeto

```
diario-humor/
├── index.html              # Página de login
├── diario.html         # Página principal do diário
├── css/
│   ├── index.css           # Estilos da página de login
│   └── diario.css          # Estilos da página do diário
├── js/
│   ├── verificações.js     # Validações de login e saída
│   ├── fazerEmoji.js       # Criação dos emojis e lógica de seleção
│   └── salvarMensagem.js   # Salvamento e carregamento das entradas
└── README.md
```
